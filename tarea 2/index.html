HTML 5 Y CSS 3

HTML 5

El 28 de Octubre de 2014 se aprobo la recomendacion HTML 5, culminando asi 10 años de trabajo. En 2004 formo el W3C retomo el trabajo el HTML y 7 años despues se alcanzo el bojetivo

Pero el HTML no era mas que el primer paso. En noviembre de 2016 se aprobo la recomendacion HTML 5.3 para finales del 2018.

CSS 3

Desde la aprobacion de CSS 2 en 1998, el W3C ha estado trabajando en la siguiente generacion de las hojas de estilo, las CSS 3. Las recomendaciones CSS 3 se plantificaron como un consejo de recomendaciones en vez de como un documento unico. El proceso resulto mucho mas largo dello esperado: Las primeras recomendaciones CSS 3 no se aprobaron hasta el 2011

Aun que los navegadores admiten muchas caracteristicas de los borradores CSS 3, la publicacion de HTML 5 no ha acelerado el desarrollo de CSS 3.
